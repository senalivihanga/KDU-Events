KDU Events LK

KDU Events LK is a university group project developed to manage and display university events in an online platform.

🎯 Project Purpose
To provide a centralized system for managing university events such as registrations, announcements, and event details.

🛠️ Technologies Used
- HTML
- CSS
- JavaScript

✨ Features
- Event listing page
- Event details display
- User-friendly UI design
- Responsive layout
- Simple navigation system
